Create a twitterbot with Scala and Akka

Created by Robert van Rijn

You do need to be careful when writing a bot that you don't go against any of Twitter's spam policies.

They're listed in the Developer Rules of the Road: Developer Rules

What are we going to do?

Getter information from RSS feeds

Shorten the RSS feed urls

Send messages to the tweet to the akka-twitter server

Receive and handle messages we receive from the server or mayve from others

Libraries we will use:

Twitter4J is an unofficial Java library for the Twitter API. With Twitter4J, you can easily integrate your Java application with the Twitter service. Twitter4J is an unofficial library.

Libraries we will use:

ROME is a set of RSS and Atom Utilities for Java that is open source under the Apache 2.0 license. ROME includes a set of parsers and generators for the various flavors of syndication feeds, as well as converters to convert from one format to another.
