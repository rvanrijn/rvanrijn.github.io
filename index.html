Step by Step Guide.

  • Setting up authorization.
  • Go to https://dev.twitter.com/apps and click on the button that says “Create a new application” (of course, you’ll need to log in with your Twitter username and password in order to do this)
  • Fill in the name, description and website fields. Don’t worry too much about this: put in whatever you like for the name and description (e.g. “My example application” and “Tutorial app for me”). For the website, give the URL of your Twitter account if you don’t have anything better to use.
  • A new screen will come up for your application. Click on the button at the bottom that says “Create my access token”.
  • Click on the “OAuth tool” tab and you’ll see four fields for authentication which you need in order to use twitter4j to access tweets and other information from Twitter: Consumer key, Consumer secret, Access token, and Access token secret.

Libraries we will use:

Twitter4J is an unofficial Java library for the Twitter API. With Twitter4J, you can easily integrate your Java application with the Twitter service. Twitter4J is an unofficial library.
